

# Library Inventory Manager – Python Lab Assignment 03 #

Author: pawan chauhan

The Library Inventory Manager is a Python-based command-line application designed to efficiently manage a small library’s book collection. It provides functionalities such as adding books, issuing and returning books, searching the catalog, and maintaining records using JSON storage. Built with clean modular design and robust error-handling.


# Project Overview #

This project showcases:

Object-Oriented Programming through well-structured Book and Inventory classes

JSON file handling for persistent data storage

Exception handling to manage missing or corrupted data files

Logging mechanisms to track execution and simplify debugging

Interactive CLI menu for seamless user operation

Organized Python package structure for maintainable code



# Key Features #

✔ Add new books (Title, Author, ISBN)
✔ Issue and return books
✔ View complete book inventory
✔ Search books by title or ISBN
✔ Persistent data saved in books.json
✔ Automatically handles missing or damaged data files
✔ Integrated logging using Python’s logging module
✔ Clean, modular, and scalable code architecture
